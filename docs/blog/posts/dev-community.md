# Introducing ATLAS Developer Community

We are excited to announce the launch of the ATLAS Developer Community, a new platform designed to bring together developers, engineers, and enthusiasts who use our APIs. 

<!-- more -->

## What's New?

[ATLAS Developer Community](https://github.com/atlas-dev-hub) is now live on Github. It's the home for any community-contributed examples or projects that utilize our APIs. Whether you're building a custom dashboard, creating a new tool, or sharing your insights, the ATLAS Developer Community is the place to showcase your work and collaborate with others.

[Software Products](https://github.com/Software-Products) continues to house all of our open source products - the Stream API, Protocol definitions, Common Utils and more.

[ATLAS Docs](https://github.com/mat-docs) remains the home for all of our official documentation, API references and curated examples. 

Each organization has its own unique purpose, and we encourage you to explore both to find the resources that best suit your needs.

## Get Involved

We've already seen some amazing projects built with our APIs. The [Configuration API Management Dashboard](https://github.com/atlas-dev-hub/example-config-api-management-dashboard) provides a visual dashboard for managing and switching between multiple SM instances, routing configuration and API calls from a single place. The [Streaming Setup](https://github.com/atlas-dev-hub/example-stream-api-kafka-setup) is a minimal Kafka-powered playground that pairs the Stream API with a Jupyter notebook example showing how to stream telemetry data into the ATLAS ecosystem.

If you've built something cool with our APIs, we want to see it! Share your projects on the ATLAS Developer Community Github Org and connect with other developers who are passionate about building with ATLAS.

[Visit the ATLAS Developer Community on GitHub ->](https://github.com/atlas-dev-hub)
