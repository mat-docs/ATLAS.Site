# ATLAS Viewer Release: v11.5.2.481-W19

This release adds a Recorder Connection Indicator improvement and resolves scatterplot session management, USB Logger stability, missing out-lap detection, and multi-session lap timing issues.

## New Features

### Recorder Connection Indicator

The recorder connection indicator has been enhanced to reflect the live link status of connected hardware units, giving users clear visual feedback on whether a recorder is connected and ready before recording begins.

The recorder border colour in the Session Browser, Recorder Editor, and Status toolbar now conveys both the recorder state and its link status:

| Recorder State | Link Status | Border Colour | Meaning |
|---|---|---|---|
| Idle | Link NOT OK | Orange | Recorder is waiting but has no active connection to the unit |
| Idle | Link OK | Green | Recorder is waiting with a confirmed connection to ADS or the unit |
| Monitor | Any | Dimmed | Recorder is in monitor mode |
| Recording | Any | Red (flashing) | Recorder is actively recording |

Additional behaviour introduced:

- **Automatic stop on link loss** — if a recorder loses its link while a manual recording is in progress, the recording is stopped automatically and an error message is shown.
- **Invalid configuration detection** — if a recorder remains Idle after an attempted state change and the link is not OK, an error is raised: `Recorder '{name}' Offline` for Ethernet and DST recorders, or `{name} configuration is invalid` for other types.

Auto-record behaviour is unaffected; link status is an additional informational layer only.

### USB Logger Conversion Improvements (#7353)

Improved the performance of the USB Logger Recorder when converting .bin files to .ssn2. 

## Bug Fixes

### Scatterplot Z‑Axis Colour Bar Missing 

Fixed an issue where scatterplots with a Z‑axis colour bar saved on a non‑active page would reopen without the colour bar once data had loaded; displays now initialise fully on page load without requiring a manual refresh.

### Event Display Crash Due to Unsortable Header Columns (#7021)

Fixed a crash that occurred when loading an event display with corrupted or invalid header column sort definitions.

### USB Logger Recorder – Crash When Drive Is Unavailable

Fixed a crash that occurred when USB Logger Recorder searched for files on a drive that no longer exists or has become inaccessible.

### Statistics Calculation Failures in Waveform (#7082)

Fixed an issue where ATLAS Statistics failed to display values for parameters in the Waveform display when any PDA (data provider) threw an exception.

### Compressed Timeline and Incorrect Lap Times (#7293)

Fixed an issue where, after several recordings, the timeline would become compressed with the current lap occupying the majority of the view and all previous laps squeezed to the left.

### Incorrect Lap Timing and Naming in Stream Recorder

Fixed an issue where lap state was not tracked per session, causing incorrect lap numbers and timing when multiple sessions were open simultaneously.
