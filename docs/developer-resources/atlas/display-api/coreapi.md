# Core API

The display API shares some core functionality with ATLAS internal APIs

- `SignalBus`
- Diagnostics
    - `ILogger`
- MVVM helpers
    - `BindableBase`
    - `DisposableBindableBase`
    - `DelegateCommand`
- IDisposable helpers
    - `DisposableBase`
- Colours
    - `AtlasColor`
    - `AtlasColors`
