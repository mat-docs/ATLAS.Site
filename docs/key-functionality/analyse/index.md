# ATLAS Analyse

Visualise, compare, and understand high‑frequency telemetry — live or post‑run.

## [ATLAS Viewer](viewer/index.md)

ATLAS Viewer provides a powerful environment for inspecting high‑resolution time‑series data. Built for motorsport engineers, it enables confident, rapid decision‑making through rich visualisation tools and flexible customisation.

### Key Functionality
- Advanced visualisation and layout customisation
- Alarms for rapid event recognition
- Optimised live data viewing
- MATLAB integration
- Compare‑set analysis
- 19 built‑in display types
- Functions and virtual channels
- Secure and reliable data handling

### Available Add‑ons
- NASCAR Telemetry Recorder
- Telemetry File Recorder
- TCP/IP Recorder
- .NET & MATLAB Functions
- MATLAB Session Access
- Ethernet Recorder
