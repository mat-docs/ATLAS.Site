# ATLAS Integrate

Extend ATLAS capabilities by connecting with third‑party systems and custom workflows.

## [Remote Telemetry Architecture (RTA)](rta/index.md)

RTA provides low‑latency access to live and historical telemetry using your existing databases. It’s designed for teams building integrated data pipelines without complex setup.

### Key Functionality

- Database integration
- Live & historical session access
- OAuth2.0 with MFA
- Microservice‑based data delivery

## [APIs](apis/index.md)

ATLAS APIs provide direct programmatic access for automation, custom tooling, and deeper analysis.

**Includes:**

- SQL Race API — query and analyse telemetry datasets
- Automation API — streamline repetitive or operational tasks
- Display API — create specialised custom visual displays
