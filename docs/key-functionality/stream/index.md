# ATLAS Stream

Distribute live telemetry seamlessly across multi‑client workflows.

## [Open Data Package](open_streaming_architecture/index.md)

Provides open access to ATLAS data streams through standardised formats, enabling collaboration and integration with external systems and bespoke engineering tools.

### Key Functionality

- Modern microservice architecture
- Open telemetry data formats
- Lightweight processing libraries
- Virtual parameter calculation

## [Data Server Package](ads/index.md)

Enables high‑performance distribution of live telemetry across multiple clients via Ethernet or Wi‑Fi. Designed for distributed teams working collaboratively.

### Key Functionality

- Daisy‑chain server setups
- Integration with Control Limited workflows
- Multicast to multiple ATLAS clients
- Coverage cursor for data integrity checks
