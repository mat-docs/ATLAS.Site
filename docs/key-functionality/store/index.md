# ATLAS Store

Organise and retain vast telemetry datasets in a structured, scalable format.

## SQL Race Management Studio (SRMS)

SRMS streamlines the storage and access of racing telemetry. Designed for teams handling large data volumes, it enables efficient session organisation, import automation, and secure database‑backed retention.

### Key Functionality
- Session management tools
- Centralised data storage with user controls
- Bulk import
- Auto‑import for routine ingestion
