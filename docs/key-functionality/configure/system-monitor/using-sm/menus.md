# Menus

### File Menu

- **Open**: Load a previously saved desktop configuration.
- **Save**: Save the current desktop layout and settings.
- **Print**: Print the contents of the active window.
- **Exit**: Close the System Monitor application.

### Edit Menu
- **Copy / Paste**: Standard clipboard operations for parameters and window contents.
- **Undo / Redo**: Revert or reapply recent changes.

### View Menu

- **Zoom In / Out**: Adjust the scale of graphical windows.
- **Grid Toggle**: Show or hide grid overlays in measurement windows.
- **Window Arrangement**: Cascade or tile open windows.

### ECU Menu

- **Connect / Disconnect**: Establish or terminate communication with the ECU.
- **Send All Parameters**: Transmit all editable parameters to the ECU.
- **Receive All Parameters**: Retrieve all parameters from the ECU.
- **Unlock Parameters**: Access restricted parameters if permitted.

### Sensors Menu

- **Sensor Status**: Display current sensor readings and statuses.
- **Sensor Configuration**: Modify sensor setup and calibration.

### Logging Menu
- **Start / Stop Logging**: Begin or end data logging sessions.
- **Log Settings**: Configure logging parameters such as sample rate and file format.
- **Export Log**: Save logged data to external formats.

### Tools Menu

- **Parameter Explorer**: Browse and filter available parameters.
- **Math Operators**: Apply mathematical functions to parameters.
- **Virtual Parameters**: Create derived parameters using formulas.

### Data Wizard Menu

- **Launch Data Wizard**: Open the Data Wizard tool for advanced analysis.
- **Import / Export**: Transfer data between System Monitor and Data Wizard.

### Desktop Menu

- **New Window**: Create a new measurement or editable window.
- **Window Properties**: Adjust layout, colours, and axis settings.
- **Page Management**: Organise windows into pages for easier navigation.
