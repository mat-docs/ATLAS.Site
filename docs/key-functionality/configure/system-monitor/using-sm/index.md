# System Monitor

System Monitor is a software tool used for real-time data monitoring, parameter editing, and logging in motorsport applications. It interfaces with ECUs and other data sources to provide engineers with flexible visualisation and control.

## Features

- Real-time acquisition and display of measurement data
- Editing of calibration parameters
- Customisable desktop layout with multiple window types
- Logging and export of data
- Integration with Data Wizard and other analysis tools
