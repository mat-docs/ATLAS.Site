# System Monitor Overview

The System Monitor is a data display tool designed to provide feedback during tuning by visualising Measurement Parameters. While it offers basic display capabilities, more advanced analysis is available through the [ATLAS visualiser](../../../../visualise/atlas/index.md).

System Monitor presents data using various display windows, each tailored to different visualisation needs. These windows are configured to show live data from Measurement Parameters, and can be customised extensively via the Window Properties dialog.

## Display Types

System Monitor supports multiple display formats to suit different data visualisation requirements. Each format uses a dedicated window type:

### [Numeric](numeric.md)
Displays one or more Measurement Parameters in rows with descriptive text.

### [Big Number](numeric.md/#big-number-display)
A variant of the Numeric display using a large font. Created automatically when a single parameter is selected via Parameter Explorer.

### [Bar Graph](bar.md)
Displays parameters as vertical or horizontal bars. Bar height and colour reflect parameter values and warning limits.

### Dial Gauge
Displays a single parameter on a dial-style gauge.

### [Oscilloscope](oscilloscope.md)
Plots parameters against time, supporting multiple trace types (Linear, Digital, Logical).

### [Grid](grid.md)
A spreadsheet-style layout allowing placement of text and parameter attributes in cells. Supports editable parameters and custom formatting.

### [Signal Sampling](signal.md)
Provides functionality to sample or zero one or more input signal parameters.
