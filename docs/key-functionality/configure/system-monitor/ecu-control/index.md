# ECU Control System Overview

The ECU Control System Monitor allows programming and monitoring of the ECU via the ECU Menu commands. The program is loaded into System Monitor during the creation of a Program Version. Security features in the Customer Base ensure compatibility between the program and the ECU.

## Key Features

- **ECU Bar**: Displays the status of the ECU.
- **Unit Status**: Shows data versions and configuration files in System Monitor and the ECU units. Supports upload/download of data.
- **ECU Search**: Provides detailed information about the ECU.
- **Kill Engine**: Stops or prevents engine start via the Tools menu.
- **Set ECU Time and Date**: Synchronises ECU time with the PC.
- **Read RAM**: Accesses detailed ECU status information.
